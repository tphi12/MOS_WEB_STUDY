# MOS_WEB_STUDY
Web for study MOS WORD

## Frontend

MOS Word Learning Hub được dựng trong thư mục `FE` bằng Vite + React + TypeScript.

```bash
cd FE
npm install
npm run dev
```

Các tính năng chính:

- Sidebar lộ trình MOS Word theo 3 nhóm: học tập/nghiên cứu, hành chính, kỹ thuật nâng cao.
- Bài học dạng checklist thực hành, có tiến độ theo từng bài.
- Smart search để tra lỗi, bài học và phím tắt.
- Quick tools cho phím tắt, lỗi thường gặp và file thực hành đề xuất.
- Copy nhanh thông số Word và flashcard tự kiểm tra.
